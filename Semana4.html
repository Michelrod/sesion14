Clase 7 - Martes 11 de abril del 2022

En clase vimos un poco más sobre CSS y aprendimos a dale color a las cajas y como dividir o distrivuir espacios con css.

CSS

Con CSS podemos crear reglas que sirven para modificar la apariencia de contenido como: Fuente, color de texto, tamaño de texto, posición.
Aprendimos que en CSS todo está dentro de una caja. Y cada caja engloba un elemento o agrupación de elementos. También aprendí sobre la estructura de una regla.

DOCUMENTO

Se hablo del selector y de la declaración. Al igual que la importancia de las (llaves) {}, los (dos puntos) : y el (punto y coma) ;.

Dentro de la declaración tenemos tenemos las propiedades que indican el aspecto a cambiar y el valor especifica las configuraciones de la propiedad.

DOMINIO

Es un nombre único asociado a una dirección IP y termina una dirección web.El DNS hace la transformación para encontrar el IP.
El profe nos dio recomendación como elegir nuestro dominio:

  • Nombre fácil de recordar y escribir.
  • Extensión formal:.com, .net, .org.
  • Nombre que te haga diferente a los demas.

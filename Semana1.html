Semana 1

Conoce un poco de mi primera semana
en laboratorio de innovación 2.

Clase 1 - Martes 21 de marzo del 2022

Esta primera clase conocimos al proffesor y a nuestros compañeros. Tuvimos una introducción al curso, donde aprendimos que HTML y CSS no son lenguajes de programación.

HTML= estructura
CSS= visual - estético

También vimos los temas que se tocaran en el curso y los dos proyectos que debemos entregar, creació de página web (personal) y un proyecto grupal.

HTML

HTML no es un lenguaje de programación, supropio nombre nos da la respuesta. Es un lenguaje de marcado o de marcas, compuesta por etiquetas que se emplean para crear la estructura completa de una página web.

CSS

Este tampoco es un lenguaje de programación sino una tecnología que se emplea para darle estilo a una página web. Gracias a CSS se puede hacer una separación entre el contenido y estructura de la web y su parte visual o estética.

Clase 2 - Miércoles 22 de marzo del 2022

La segunda clase fue un poco más de teorica. Llegamos a manejar en modod de practica el html como modificar algún template. No fue tan complicado, lo unico tedioso, al menos para mi, fue encontrar exactamente lo que se tenia que modificar.
Aprendimos a modificar textos y agregar imagenes.
También se hbalo sobre la bitácora, que va dentro de ella y como armarla.

HTML

Profundizamos más sobre HTML.
Sabiendo que no es un lenguaje de programación es una serie de elementos que usaremos para encerrar las diferentes partes o etapas del contenido, para que se vean o se comporten de una determinada manera y tengan una funsionalidad.
También vimos como es una estructura de documento HTML y sus etiquetas.
